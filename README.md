3D CSS Cube Loader

A lightweight and visually refined 3D cube loader built using pure HTML and CSS. This project demonstrates advanced front-end techniques such as 3D transforms, keyframe animations, and glassmorphism styling without relying on JavaScript.

Demo

https://github.com/Mithanya/pure-css-3d-loader/blob/main/3d%20loder.mp4

Features
Fully 3D cube constructed using perspective and transform-style: preserve-3d
Smooth multi-axis rotation across X, Y, and Z axes
Independent glowing faces with staggered animation timing
Lightweight and performance-focused with no JavaScript
Maintains smooth performance across modern devices
Responsive and adaptable to different screen sizes
Modern glassmorphism-inspired visual design
Technologies Used
HTML5
CSS3 (3D transforms, keyframe animations, flexbox, gradients, shadows)
How It Works

The cube is created using six elements representing each face. Each face is positioned in 3D space using translateZ and rotation transforms. A parent container applies perspective to create depth and uses transform-style: preserve-3d to maintain the three-dimensional structure.

A continuous keyframe animation rotates the cube across multiple axes, producing a smooth and dynamic motion. Each face includes a glow effect using box-shadow with staggered animation delays, creating a subtle wave-like lighting effect.
